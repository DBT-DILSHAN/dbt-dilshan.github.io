# Tharindu Dilshan - Portfolio

Welcome to my professional portfolio website! I'm an **Industrial Electrical Technician** and **Electrical Installation & Maintenance Specialist** based in Sri Lanka.

## 🌟 Features

- **Modern Design**: Sleek glassmorphism UI with smooth animations
- **Dark/Light Theme**: Toggle between dark and light modes seamlessly
- **Fully Responsive**: Optimized for all devices (mobile, tablet, desktop)
- **SEO Optimized**: Includes meta tags, sitemap, and robots.txt
- **Interactive Elements**: Smooth scroll behavior, progress indicators, and animated components
- **Accessible**: Supports reduced motion preferences and high contrast modes
- **Performance Ready**: Optimized with font preconnection and efficient animations

## 📋 Sections

- **About Me**: Professional background and expertise
- **Skills**: Industrial electrical systems, VFD, Motor Control, Panel Board installation
- **Experience**: Career highlights and achievements
- **CV**: Downloadable resume (PDF)
- **Contact**: WhatsApp integration for quick communication

## 🛠️ Technology Stack

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS variables and animations
- **JavaScript**: Interactive features and theme switching
- **Font Awesome**: Icon library
- **AOS (Animate On Scroll)**: Scroll-triggered animations
- **Google Fonts**: Poppins, Space Mono, and Outfit typefaces

## 🎨 Design System

### Color Palette
- **Primary**: Green accent (#10B981, #34D399)
- **Dark Theme**: Black background (#000000)
- **Light Theme**: White background (#FFFFFF)
- **Text**: White on dark, Near-black on light

### Typography
- **Display Font**: Outfit
- **Body Font**: Poppins
- **Monospace**: Space Mono
